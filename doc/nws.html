Kit de connection Numworks

Attention, Numworks va verrouiller ses calculatrices à partir de la version 16 de son logiciel (prévue pour la rentrée 2021) ce qui les rendra incompatibles avec Xcas et tous les logiciels développés par la communauté. Ne faites pas de mise à jour, car c'est irréversible. Cette page contient un kit de connexion avec votre calculatrice qui vous évitera d'aller sur le workshop de Numworks et de faire une mise à jour par inadvertance.
  • : importe un script Python de la calculatrice vers l'ordinateur. Vous pouvez aussi faire un , c'est-à-dire une sauvegarde complète de tous les scripts de la calculatrice.
  • : si le fichier a pour extension .py on envoie un script Python de l'ordinateur vers la calculatrice. Si le fichier a pour extension .nws, on remplace le contenu de la calculatrice par la sauvegarde.
  • Vous pouvez aussi échanger des scripts avec la calculatrice depuis Xcas pour Firefox , les modifier, les tester, et les partager sous forme de liens par email ou sur des forums.
  • : installe Xcas sur votre calculatrice Numworks.
  • : Assistance si votre calculatrice est bloquée.
    Connectez la calculatrice, appuyez sur la touche 6 de la calculatrice, enfoncez un stylo dans le bouton RESET au dos en laissant la touche 6 appuyée, relachez la touche 6, l'écran doit etre éteint et la diode rouge allumée.
    Si le message http://... tente de se connecter apparait, cliquez sur STM32 BOOTLOADER associé puis sur le bouton Connexion.
  • Lutte contre la fraude : pour certifier que le firmware d'une calculatrice Numworks N0110 est une version non modifiée de Delta+KhiCAS, cliquez sur l'un des deux boutons

    Le premier test prend 15 secondes environ, le deuxième est plus sécurisé mais dure 1 minute environ. Ce test peut etre fait par un surveillant pendant un examen depuis son smartphone depuis ce lien avec un cable smartphone-Numworks.
Cette page fonctionne uniquement avec un navigateur compatible webusb, par exemple Chromium ou Chrome et depuis un serveur web. Firefox n'a pas de support pour webusb pour des raisons de sécurité, si vous avez des craintes à ce sujet, vous pouvez connecter votre calculatrice Numworks avec Xcas (version Windows, Linux et Mac).
Pour les experts souhaitant utiliser cette page en local, activez un serveur web local, par exemple en tapant dans un Terminal la commande python3 -m http.server depuis le répertoire où vous avez décompressé xcashtml.zip, puis cliquez sur ce lien.
