Outils Numworks


  • Si votre calculatrice est bloquée, vous pouvez installer Xcas puis dans le menu Fich, sous-menu Numworks, choisissez Assistance calculatrice bloquee pour installer le mode de récupération sur la Numworks.
  • Pour certifier que le firmware d'une calculatrice Numworks N0110 est une version non modifiée de Delta+KhiCAS, cliquez sur l'un des deux boutons

    Le premier test prend 15 secondes environ, le deuxième est plus sécurisé mais dure 1 minute environ.
Cette page fonctionne uniquement avec un navigateur compatible webusb, par exemple Chromium ou Chrome et avec un serveur. Pour une utilisation en local, il faut activer un serveur, par exemple avec la commande python3 -m http.server, puis cliquer sur ce lien.
