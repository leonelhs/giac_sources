6.59.4  Liste des variables et des expressions algébriques : algvar

algvar a comme paramètre une expression.
algvar renvoie un vecteur de composantes le nom des variables symboliques, par ordre d’extension algebriques, utilisées dans cette expression.
On tape :

algvar(y+x*sqrt(z))

On obtient :

[[y,x],[z]]

On tape :

algvar(y*sqrt(x)*sqrt(z))

On obtient :

[[y],[z],[x]]

On tape :

algvar(y*sqrt(x*z))

On obtient :

[[y],[x,z]]

On tape :

algvar(y+x*sqrt(z)+y*sin(x))

On obtient :

[[x,y,sin(x)],[z]]
