6.14.22  Pour substituer plus simplement une valeur à une variable dans une expression : ()

Lorsque l’on veut donner dans une expression de plusieurs variables (par exemple Expr:=x+2y+3z) une valeur à certaines variables (par exemple x,y) on peut le faire de différentes façons. On tape par exemple xpr:=x+2y+3z, puis : On utilise subst, on tape :

  • subst(subst(Expr,x=1),y=2)
  • subst(Expr,[x=1,y=2])
  • subst(Expr,[x,y],[1,2])

On n’utilise pas subst mais une écriture simplifiée proche d’une écriture fonctionnelle :

  • Expr(x=1,y=2)
  • Expr([x,y],[1,2])

On obtient dans tous les cas :
5+3*z Autre exemple
On tape :
(h*k*t^2+h^3*t^3)(t=2)
On obtient :
4*h*k+8*h^3
