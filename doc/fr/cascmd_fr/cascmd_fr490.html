6.40.3  Union de deux ensembles ou de deux listes : union

union renvoie l’ensemble qui est l’union de deux ensembles ou de deux listes.
union est un opérateur infixé.
On tape :

set[1,2,3,4] union set[5,6,3,4]

Ou on tape :

% {1,2,3,4% } union % {5,6,3,4% }

On obtient :

[1,2,3,4,5,6]

On tape :

[1,2,3] union [2,5,6]

On obtient :

[1,2,3,5,6]
