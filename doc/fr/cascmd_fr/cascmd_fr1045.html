11.13.7  Savoir si 4 points sont cocycliques : is_concyclic est_cocyclique

Voir aussi : 10.18.3 pour la géométrie plane.
est_cocyclique est une fonction booléenne ayant comme argument une liste ou une séquence de points.
est_cocyclique vaut 1 si les points sont cocycliques, et vaut 0 sinon.
On tape :

est_cocyclique([2,0,0],[0,2,0],[sqrt(2),sqrt(2),0], [0,0,2],[2/sqrt(3),2/sqrt(3),2/sqrt(3)])

On obtient :

1

On tape :

est_cocyclique([2,0,0],[0,2,0],[1,1,0],[0,0,2],[1,1,1])

On obtient :

0
