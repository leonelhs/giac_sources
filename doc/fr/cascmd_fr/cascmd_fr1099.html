12.1.2  Écriture d’un fichier wav : writewav

La commande writewav effectue l’écriture d’une donnée de type son dans un fichier au format WAV sur le disque dur. Elle est donc inverse de la commande readwav.
On passe en argument le nom du fichier et les données son.
On peut soit utiliser le même format que celui renvoyé par la commande readwav, soit pour un son mono utilisant les valeurs des paramètres par défaut (16 bits, 44100 Hz) donner uniquement la liste des données numériques du son sur le canal.
Par exemple la commande :
writewav("la.wav",2^14*(sin(2*pi*440*soundsec(1))))
sauvegarde dans le fichier de nom la.wav, un son pur de fréquence 440 Hz (un la), échantilloné 44100 fois par seconde.
