6.34.13  PGCD dans ℤ/pℤ[x] : gcd

Lorsque gcd a deux polynômes à coefficients dans ℤ/pℤ comme arguments (p doit être premier).
gcd calcule le PGCD des deux polynômes dans ℤ/pℤ[x] (voir aussi 6.28.7 pour les polynômes à coefficients non modulaires).
On tape :

gcd((2*x^2+5)% 13,(5*x^2+2*x-3)% 13)

On obtient :

(-4% 13)*x+5% 13

On tape :

gcd(x^2+2*x+1,x^2-1) mod 5

On obtient :

1

Mais si on tape :

gcd((x^2+2*x+1,x^2-1)) mod 5)

gcd est calculé dans ℤ[X] puis le calcul modulaire est effectué, on obtient :

x% 5
