6.53.10  Base de l’intersection de deux sous espaces vectoriels : ibasis

ibasis a comme argument deux listes de vecteurs qui engendrent deux sous espaces vectoriels de ℝn.
ibasis renvoie une liste constituée de vecteurs formant une base de l’intersection de ces sous espaces vectoriels.
On tape :

ibasis([[1,2]],[[2,4]])

On obtient :

[[1,2]]
