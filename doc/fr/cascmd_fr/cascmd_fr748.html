8.4.5  Fonction de répartition de la loi binomiale négative : negbinomial_cdf

Lorsqu’une variable aléatoire X suit une loi binomiale négative NegBin(n,p), on a :
negbinomial_cdf(n,p,x)=Proba(X ≤ x)=
negbinomial(n,0,p)+...+negbinomial(n,floor(x),p).
negbinomial_cdf(n,p,x,y)=Proba(x≤ X ≤ y)=
negbinomial(n,ceil(x),p)+...+negbinomial(n,floor(y),p).
On tape :

negbinomial_cdf(4,0.5,2)

On obtient :

0.34375

On peut vérifier que :

negbinomial(4,0,0.5)+negbinomial(4,1,0.5)+negbinomial(4,2,0.5)
0.34375

On tape :

negbinomial_cdf(2,0.3,1)

On obtient :

0.216

On tape :

negbinomial_cdf(2,0.3,3)

On obtient :

0.47178
