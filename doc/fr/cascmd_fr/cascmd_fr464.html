6.36.5  Résidu d’une expression en un point : residue

residue a comme argument une expression dépendant d’une variable, le nom de cette variable et un complexe a ou bien une expression dépendant d’une variable et l’égalité : nom_de_variable=a. residue renvoie le résidu de cette exppression au point a.
On tape :

residue(cos(x)/x^3,x,0)

Ou on tape :

residue(cos(x)/x^3,x=0)

On obtient :

(-1)/2

On tape :

int(exp(i*t)/(2*exp(i*t)-1),t=0..2*pi)

On obtient :
Searching int of 1/(2*t−1) where t is on the unit circle, using residues

(2*pi)/2

On tape :

int(exp(2*i*t)/(2*exp(i*t)-1))^2,t=0..2*pi)

On obtient :
Searching int of t/(4*t2−4*t+1) where t is on the unit circle, using residues

(2*pi)/4
