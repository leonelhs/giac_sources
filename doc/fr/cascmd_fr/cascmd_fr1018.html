11.9.4  Ligne polygonale dans l’espace : open_polygon polygone_ouvert

Voir aussi : 10.12.4 pour la géométrie plane.
polygone_ouvert, en géométrie 3-d, a comme argument une séquence) de n points 3-d.
polygone_ouvert renvoie et trace la ligne polygonale ayant pour sommets ces n points.
On tape :

polygone_ouvert(point(0,0,0),point(0,1,0),point(0,2,2),point(1,0,2))

On obtient :

Une ligne polygonale de sommets les points donnés
