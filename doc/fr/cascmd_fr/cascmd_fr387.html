6.28.29  Convertir une liste en polynôme de n variables : convert convertir

convert, avec l’option polynom, permet de convertir une liste en un polynôme de n variables.
On tape :

l:=[[1,[2,0]],[2,[1,1]],[1,[0,2]]]
p:=convert(l,polynom)

On obtient :

%%%1,[2,0]%%%+%%%2,[1,1]%%%+%%%1,[0,2]%%%

On tape :

poly2symb(p,[x,y])

On obtient :

x^2+2*x*y+y^2
