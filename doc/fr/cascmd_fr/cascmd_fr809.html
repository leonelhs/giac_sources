9.4.7  Fonction testant le type de son argument: compare

compare est une fonction qui a deux arguments et qui renvoie 1 si ses arguments ont des types différents ou ou si ses arguments sont de même type et sont mis dans l’ordre croissant, et qui renvoie 0 sinon.
On tape :

compare(1,2)

On obtient :

1

On tape :

compare(2,1)

On obtient :

0

On tape :

compare("3","a")

On obtient :

1

On tape :

compare("a",3)

On obtient :

0

On tape :

compare(3,"a")

On obtient :

1

On tape :

compare("a",3)

On obtient :

0

En effet on a : type(3)=DOM_INT=2 et type("a")=DOM_STRING=12
