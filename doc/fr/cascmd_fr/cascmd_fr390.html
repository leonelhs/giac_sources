6.28.5  Reste euclidien : Rem

Rem est la forme inerte de rem.
Rem renvoie le reste de la division euclidienne de deux polynômes (division selon les puissances décroissantes) sans l’évaluer et cela permet de calculer le reste euclidien de deux polynômes à coefficients dans ℤ/pℤ en utilisant la syntaxe Maple.
Attention il faut être en mode Maple pour que cela soit efficace.
On tape, en mode Xcas :

Rem(x^3-1,x^2-1)

On obtient :

rem(x^3-1,x^2-1)

On peut aussi taper pour avoir le reste de x2+2x+4 par x2+x+2 :

Rem([1,2,4],[1,1,2])

On obtient :

rem([1,2,4],[1,1,2])

En mode Maple on tape :

Rem(x^3+3*x,2*x^2+6*x+5) mod 5

On obtient :

2*x

La division est faite dans ℤ/5ℤ[X] alors que pour :

rem(x^3+3*x,2*x^2+6*x+5) mod 5

la division est faite dans ℤ[X] puis est réduite après :

12*x

Si Xcas n’est pas en mode Maple, la division des polynômes dans ℤ/pℤ[X] se fait en tapant :

rem((x^3+3*x)% 5,(2x^2+6x+5)%5)
