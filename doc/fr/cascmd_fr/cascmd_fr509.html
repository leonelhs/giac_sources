6.42.7  Supprimer l’élément ’indice donné dans une liste ou une chaine de caractères : suppress

suppress supprime dans une liste l’élément d’indice donné.
Attention ! l’indice du premier élément est 0.
On tape :

suppress([3,4,2],1)

On obtient :

[3,2]

On tape :

suppress("342",1)

On obtient :

"32"
