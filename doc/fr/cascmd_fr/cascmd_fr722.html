8.2.17  Graphe de la régression polynomiale :
polynomial_regression_plot

Pour approcher les données par le grahe d’une fonction polynômiale de degré ≤ n d’équation y=a0xn+..+an, on utilise polynomial_regression_plot.
polynomial_regression_plot a les mêmes premiers arguments que covariance, le dernier argument étant le degré du polynôme renvoyé.
On tape :

polynomial_regression_plot([[1.0,1],[2,4],[3,9],[4,16]],3)

On obtient :

Le graphe de la fonction polynômiale de degré ≤ n d’équation y=x2

car c’est la fonction polynômiale d’équation y=1*x2 qui approche au mieux les données.
