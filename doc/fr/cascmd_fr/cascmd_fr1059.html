11.16.1  Le cube : cube

cube a comme argument trois points A,B,C.
cube dessine le cube direct de coté AB dont une face est dans le plan(A,B,C).
On tape :

cube([0,0,0],[0,4,0],[0,0,1])

On obtient :

Le cube de sommets [0,0,0],[4,0,0], [0,4,0],[0,0,4]...

On tape :

cube([0,0,0],[0,4,0],[0,0,-1])

On obtient :

Le cube symétrique du précédent par rapport à 0y
