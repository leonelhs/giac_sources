8.4.20  Fonction de répartition de la loi du χ2 : chisquare_cdf

Lorsqu’une variable aléatoire X suit une loi du χ2 ayant n degrés de liberté, on a :
Proba(X ≤ x)= chisquare_cdf(n,x).
Proba(x ≤ X ≤ y)= chisquare_cdf(n,x,y).
On tape :

chisquare_cdf(5,11)

On obtient :

0.948620016517

On tape :

chisquare_cdf(3,11)

On obtient :

0.988274124422
