6.34.8  Division dans ℤ/pℤ ou ℤ/pℤ[x] : /

/ divise deux entiers dans ℤ/pℤ, ou divise deux polynômes A et B dans ℤ/pℤ[x]. Pour les polynômes, le résultat est la fraction rationnelle A/B simplifiée dans ℤ/pℤ[x].
Pour les entiers dans ℤ/pℤ, on tape :

5% 13/2% 13

On obtient :

-4% 13

puisque 2 est inversible dans ℤ/13ℤ.
Pour les polynômes à coefficients dans ℤ/pℤ.
On tape :

(2*x^2+5)% 13/(5*x^2+2*x-3)% 13

On obtient :

((6% 13)*x+1% 13)/((2% 13)*x+2% 13)
