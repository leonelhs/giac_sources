10.9.3  Le segment en géométrie plane : segment

Voir aussi : 11.5.4 pour la géométrie 3-d.
segment, en géométrie plane, a comme argument 2 points (ou 2 nombres complexes représentant l’affixe de ces points ou encore une liste de 2 points ou de 2 complexes).
segment renvoie et trace le segment défini par les deux arguments.
On tape :

segment(-1,1+i)

On obtient :

Le segment reliant le point(-1) au point(1+i)

On tape :

segment(point(-1),point(1,1))

On obtient :

Le segment reliant le point(-1) au point(1+i)

Attention
segment peut aussi dessiner un vecteur en géométrie plane (cf 10.9.4)
