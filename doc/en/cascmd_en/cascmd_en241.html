7.3.4  Composing functions

In Xcas, the composition of functions is done with the infixed operator @ (see Section 7.1.1). For example:

(sq@sin+id)(x)
     
sin2(x)+x           

The repeated composition of a function with itself several times is done with the infixed operator @@ (see Section 7.1.1). For example:

(sin@@3)(x)
