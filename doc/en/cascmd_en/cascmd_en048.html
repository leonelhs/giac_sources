4.4.6  String defined by the ASCII codes of its characters : char

char takes as argument a list l of ASCII codes.
char returns the string whose characters have as ASCII codes the elements of the list l.
Input :

char([97,98,99,100])

Output :

"abcd"

Input :

char(97)

Output :

"a"

Input :

char(353)

Output :

"a"

because:
353−256=97.
