3.17.1  Animation of a 2D graph : animate

animate can create a 2-d animation with graphs of functions depending on a parameter. The parameter is specified as the third argument of animate, the number of pictures as fourth argument with frames=number, the remaining arguments are the same as those of the plot command, see section 3.3, p. ??.
Input :

animate(sin(a*x),x=-pi..pi,a=-2..2,frames=10,color=red)

Output :

a sequence of graphic representations of y=sin(ax) for 11 values of a between -2 and 2
