4.1.7  Evaluating booleans

The Maple command evalb evaluates a boolean expression (see Section 4.1). Since Xcas evaluates booleans automatically, it includes a evalb command only here for compatibility and is equivalent to eval (see Section 8.1.1).

  • evalb takes bool, a boolean expression.
  • evalb(bool) returns 1 if bool is true and returns 0 otherwise.

Examples

evalb(sqrt(2)>1.41)

or:

sqrt(2)>1.41
     
1           
evalb(sqrt(2)>1.42)

or:

sqrt(2)>1.42
     
0
