10.9.12  Rebuilding a fraction from its value modulo p

Given an integer n and a modulus p, the fracmod (or iratrecon, for Maple compatibility) command finds the rational number equal to n (mod p ), where both the numerator and denominator are not greater than √p/2 in absolute value.

  • fracmod (or iratrecon) takes two arguments:
    • n, an integer (representing a fraction).
    • p, an integer (the modulus).
  • fracmod(n,p) (or iratrecon(n,p)) returns, if possible, a fraction a/b such that
