2.4.7  Cmds menu

The Cmds menu contains various giac functions and constants separated into categories. If Auto index help is chosen (see Section 2.5.9), then when you select a function or constant, the help index (see Section 2.3.3) opens with the function or constant selected, which can be used to insert the entry on the command line. Otherwise, the constant or function will be inserted on the command line.
