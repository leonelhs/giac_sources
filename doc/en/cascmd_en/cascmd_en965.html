25.11.10  Checking whether an object in space is an isosceles triangle

See Section 24.15.6 for checking for isosceles triangles in two-dimensional geometry.

The is_isosceles command determines whether or not a geometric object is an isoceles triangle.

  • is_isosceles takes G, a geometric object or a sequence of three points assumed to be the vertices of a triangle.
  • is_isosceles(G) returns 1,2 or 3 if the object is an isoceles triangle (the number indicates which vertex is on two equal sides), returns 4 if the object is an equilateral triangle, and returns 0 otherwise.

Examples

is_isosceles([2,0,0],[0,0,0],[0,2,0])
     
2           
T:=triangle_isosceles([0,0,0],[2,2,0],[2,2,2]); is_isosceles(T)
