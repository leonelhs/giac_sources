11.13.3  The ordinate of a point or vector in the plane: ordinate

See section 12.12.2 for ordinates in three-dimensional geometry.

The ordinate command takes a point as argument.

ordinate returns the ordinate (y-coordinate).
Input:

ordinate(point(1 + 2*i))

Output:

2

Input:

ordinate(point(i) - point(1 + 2*i))

Output:

-1

Input:

ordinate(1 + 2*i)

Output:

2

Input:

ordinate([1,2])

Output:

2
