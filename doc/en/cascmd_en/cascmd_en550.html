14.3.8  Singular value decomposition

The singular value decomposition of a matrix A is a factorization A=USQT, where U and Q are orthogonal and S is a diagonal matrix. The svd command finds the singular value decomposition of a matrix.

  • svd takes A, a numeric square matrix.
  • svd(A) returns a list [U,s,Q] where U and Q are the orthogonal matrices of the singular value decomposition and s is the diagonal of the matrix S.

You can get the diagonal matrix S from s with S=diag(s) (see Section 13.2.2).
