2.2.5  Evaluate booleans : evalb

Inside Maple, evalb evaluates an boolean expression. Since Xcas evaluates booleans automatically, evalb is only here for compatibility and is equivalent to eval
Input :

evalb(sqrt(2)>1.41)

Or :

sqrt(2)>1.41

Output :

1

Input :

evalb(sqrt(2)>1.42)

Or :

sqrt(2)>1.42

Output :

0
