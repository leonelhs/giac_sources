2.35.6  Transform a sequence into a list : [] nop

To transform a sequence into list, just put square brackets ([]) around the sequence or use the command nop.
Input :

[seq(j^3,j=1..4)]

or :

seq(j^3,j,1,4)

or :

[(j^3)$(j=1..4)]

Output :

[1,4,9,16]

Input :

nop(1,4,9,16)

Output :

[1,4,9,16]
