2.32.4  Factorization in ℤ/pℤ[x] : Factor

Factor is the inert form of factor.
Factor takes as argument a polynomial.
Factor returns factor without evaluation. It is used in conjonction with mod in Maple syntax mode to factorize a polynomial with coefficients in ℤ/pℤ where p must be prime.
Input in Xcas mode :

Factor((-3*x^3+5*x^2-5*x+4)%13)

Output :

factor((-3*x^3+5*x^2-5*x+4)%13)

you need to eval(ans()) to get :

((1%13)*x+-6%13)*((-3%13)*x^2+-5%13)

Input in Maple mode :

Factor(-3*x^3+5*x^2-5*x+4) mod 13

Output :

-3*(1*x-6)*(1*x^2+6)
