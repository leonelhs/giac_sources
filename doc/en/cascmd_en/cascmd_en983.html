13.13.7  Check if four points in space are cocyclic: is_concyclic

See section 12.16.3 for checking for concyclicity in two-dimensional geometry.

The is_concyclic command takes as argument a list or sequence of points.

is_concyclic returns 1 if the points are cocyclic, it returns 0 otherwise.

Input:

is_concyclic([2,0,0],[0,2,0],[sqrt(2),sqrt(2),0],
[0,0,2],[2/sqrt(3),2/sqrt(3),2/sqrt(3)])

Output:

1

Input:

is_concyclic([2,0,0],[0,2,0],[1,1,0],[0,0,2],[1,1,1])

Output:

0
