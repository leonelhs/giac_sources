13.5.5  Padé expansion

The pade command finds a rational expression which agrees with a function up to a given order.

  • pade takes 4 arguments
    • expr, an expression.
    • x, the variable name.
    • n, an integer or R, a polynomial.
    • p, an integer.
  • pade(expr,x,n,p) or pade(expr,x,P,p) returns a rational function P/Q such that degree(P)<p and P/Q≡ expr(mod xn+1 ) (meaning that P/Q and f have the same Taylor expansion at 0 up to order n) or P/Q≡ expr· f(mod R ), respecively.

pade(exp(x),x,5,3)
