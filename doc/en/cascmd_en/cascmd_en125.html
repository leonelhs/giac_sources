5.3.12  Concatenating two lists or a list and an element

The concat or augment command combines to lists or adds an element to a list.

  • concat takes two arguments: L1 and L2, two lists or a list and an element (in any order).
  • concat(L1,L2) returns a list consisting of the elements of L1 (or L1 itself if it isn’t a list) followed by the elements of L2 (or L2 itself).

Examples

concat([3,4,2],[1,2,4])

or:

augment([3,4,2],[1,2,4])
